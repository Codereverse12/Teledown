NAME
	Teledown - Script to download multiple Telegram Images
	
SYNOPSIS
	./teledown [OPTION] CHANNELNAME...

Description
      CHANNELNAME is the name after the protocol and hostname
      e.g 
          https://t.me/--->[CHANNELNAME] <---

Options                             Descriptions

-h                                  List of commands

-N[Number]                          [Number] of images to Download

E.g  ./teledown WallpapersETdaily 
     ./teledown -N 3 WallpapersETdaily
